# 订单排餐日历接口文档

## 概述

订单排餐日历是"按日期送"（SCHEDULE）排餐模式的可视化选择工具，允许用户通过日历界面选择送餐日期，并为每个日期配置餐次（早餐、午餐、晚餐）。

## 组件说明

### 1. MealScheduleCalendar

主日历组件，提供日期选择和餐次配置功能。

#### Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| value | v-model 绑定值，选中的日期及餐次 | Array | [] |
| startDate | 订单开始日期（最小可选日期） | String | null |
| endDate | 订单结束日期（最大可选日期） | String | null |
| readonly | 是否只读 | Boolean | false |

#### value 数据格式

```javascript
[
  {
    date: '2026-04-01',        // 日期字符串（yyyy-MM-dd）
    mealTypes: ['BREAKFAST', 'LUNCH']  // 该日期的餐次配置
  },
  {
    date: '2026-04-02',
    mealTypes: ['BREAKFAST', 'LUNCH', 'DINNER']
  }
]
```

#### Events

| 事件名 | 说明 | 回调参数 |
|--------|------|----------|
| input | 选中的日期及餐次变化时触发 | (value: Array) |
| selection-change | 选中的餐数变化时触发 | (mealCounts: Object) |

#### selection-change 回调参数格式

```javascript
{
  breakfastCount: 5,      // 早餐总份数
  lunchDinnerCount: 10,  // 午餐+晚餐总份数
  lunchCount: 6,          // 午餐份数
  dinnerCount: 4          // 晚餐份数
}
```

#### 使用示例

```vue
<template>
  <MealScheduleCalendar
    v-model="deliveryDates"
    :start-date="order.startDate"
    :end-date="order.endDate"
    @selection-change="handleSelectionChange"
  />
</template>

<script>
import MealScheduleCalendar from '@/components/Calendar/MealScheduleCalendar.vue'

export default {
  components: { MealScheduleCalendar },
  data() {
    return {
      deliveryDates: [
        { date: '2026-04-01', mealTypes: ['BREAKFAST', 'LUNCH'] }
      ],
      order: {
        startDate: '2026-04-01',
        endDate: '2026-04-30'
      }
    }
  },
  methods: {
    handleSelectionChange(mealCounts) {
      console.log('早餐份数:', mealCounts.breakfastCount)
      console.log('午晚份数:', mealCounts.lunchDinnerCount)
    }
  }
}
</script>
```

### 3. MealTypeSelector

餐次选择面板组件。

#### Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| date | 日期对象 | Date | 必填 |
| value | v-model 绑定值，已选中的餐次 | Array | [] |
| disabled | 是否禁用 | Boolean | false |

#### Events

| 事件名 | 说明 | 回调参数 |
|--------|------|----------|
| input | 餐次变化时触发 | (mealTypes: Array) |
| save | 点击保存按钮时触发 | (mealTypes: Array) |
| cancel | 点击取消按钮时触发 | - |

#### 使用示例

```vue
<template>
  <MealTypeSelector
    v-model="selectedMealTypes"
    :date="currentDate"
    @save="handleSave"
  />
</template>

<script>
import MealTypeSelector from '@/components/Calendar/MealTypeSelector.vue'

export default {
  components: { MealTypeSelector },
  data() {
    return {
      currentDate: new Date(),
      selectedMealTypes: ['BREAKFAST', 'LUNCH']
    }
  },
  methods: {
    handleSave(mealTypes) {
      console.log('保存的餐次:', mealTypes)
    }
  }
}
</script>
```

## 工具函数

`@/utils/calendar.js` 提供了日期处理工具函数。

### 主要函数

#### 1. generateCalendarData(year, month, firstDayOfWeek)

生成日历数据（包含填充日期）。

```javascript
import { generateCalendarData } from '@/utils/calendar'

const calendar = generateCalendarData(2026, 3, 0) // 2026年4月，周日起始
```

返回格式：
```javascript
[
  {
    date: Date,
    day: 1,
    isCurrentMonth: false,
    isPrevMonth: true,
    isNextMonth: false
  },
  // ...
]
```

#### 2. formatDate(date)

格式化日期为 `yyyy-MM-dd`。

```javascript
import { formatDate } from '@/utils/calendar'

const dateStr = formatDate(new Date()) // '2026-04-11'
```

#### 3. parseDate(dateStr)

解析 `yyyy-MM-dd` 格式的日期字符串。

```javascript
import { parseDate } from '@/utils/calendar'

const date = parseDate('2026-04-11')
```

#### 4. isDateInRange(date, startDate, endDate)

检查日期是否在范围内。

```javascript
import { isDateInRange, parseDate } from '@/utils/calendar'

const result = isDateInRange(
  parseDate('2026-04-11'),
  '2026-04-01',
  '2026-04-30'
) // true
```

#### 5. calculateMealCounts(deliveryDates)

根据日期和餐次配置计算餐数。

```javascript
import { calculateMealCounts } from '@/utils/calendar'

const counts = calculateMealCounts([
  { date: '2026-04-01', mealTypes: ['BREAKFAST', 'LUNCH'] },
  { date: '2026-04-02', mealTypes: ['BREAKFAST', 'LUNCH', 'DINNER'] }
])
// {
//   breakfastCount: 2,
//   lunchDinnerCount: 3,
//   lunchCount: 2,
//   dinnerCount: 1
// }
```

#### 6. normalizeDeliveryDates(value, defaultMealTypes)

将旧格式转换为新格式（兼容性处理）。

```javascript
import { normalizeDeliveryDates } from '@/utils/calendar'

// 旧格式：字符串数组
const oldFormat = ['2026-04-01', '2026-04-02']
const newFormat = normalizeDeliveryDates(oldFormat)
// [
//   { date: '2026-04-01', mealTypes: ['BREAKFAST', 'LUNCH', 'DINNER'] },
//   { date: '2026-04-02', mealTypes: ['BREAKFAST', 'LUNCH', 'DINNER'] }
// ]
```

#### 7. denormalizeDeliveryDates(deliveryDates)

将新格式转换为旧格式（兼容性处理）。

```javascript
import { denormalizeDeliveryDates } from '@/utils/calendar'

const newFormat = [
  { date: '2026-04-01', mealTypes: ['BREAKFAST', 'LUNCH'] }
]
const oldFormat = denormalizeDeliveryDates(newFormat)
// ['2026-04-01']
```

## 在 OrderForm 中的集成

OrderForm 组件已经集成了日历功能。当 `scheduleMode === 'SCHEDULE'` 时，会自动显示日历组件。

```vue
<template>
  <el-form>
    <!-- 排餐模式选择 -->
    <el-form-item label="排餐模式">
      <el-select v-model="form.scheduleMode">
        <el-option label="指定日期送" value="SCHEDULE" />
        <el-option label="每天送" value="DAILY" />
        <el-option label="周末送" value="WEEKEND" />
        <el-option label="工作日送" value="WEEKDAY" />
      </el-select>
    </el-form-item>

    <!-- 指定日期送时显示日历 -->
    <MealScheduleCalendar
      v-if="form.scheduleMode === 'SCHEDULE'"
      v-model="form.deliveryDatesWithMealTypes"
      :start-date="form.startDate"
      :end-date="form.endDate"
      @selection-change="onCalendarSelectionChange"
    />
  </el-form>
</template>

<script>
import { normalizeDeliveryDates, denormalizeDeliveryDates } from '@/utils/calendar'

export default {
  computed: {
    deliveryDatesWithMealTypes: {
      get() {
        return normalizeDeliveryDates(this.form.deliveryDates)
      },
      set(val) {
        this.$set(this.form, 'deliveryDates', denormalizeDeliveryDates(val))
      }
    }
  },
  methods: {
    onCalendarSelectionChange(mealCounts) {
      this.$set(this.form, 'breakfastCount', mealCounts.breakfastCount)
      this.$set(this.form, 'lunchDinnerCount', mealCounts.lunchDinnerCount)
      this.onMealCountChange()
    }
  }
}
</script>
```

## 数据持久化

### 保存订单时

在 `order/index.vue` 中，使用 `serializeDeliveryDates` 方法将日历数据序列化为 JSON 字符串：

```javascript
async submitForm() {
  const payload = {
    ...this.form,
    deliveryDates: this.serializeDeliveryDates(this.form.deliveryDates)
  }
  await orderApi.add(payload)
}
```

### 加载订单时

从后端加载的数据是 JSON 字符串，需要解析：

```javascript
// 后端返回的数据格式：
// { deliveryDates: '["2026-04-01","2026-04-02"]' }

// OrderForm 的 computed 会自动处理格式转换
deliveryDatesWithMealTypes: {
  get() {
    return normalizeDeliveryDates(this.form.deliveryDates)
  },
  set(val) {
    this.$set(this.form, 'deliveryDates', denormalizeDeliveryDates(val))
  }
}
```

## 注意事项

1. **格式兼容性**：工具函数 `normalizeDeliveryDates` 和 `denormalizeDeliveryDates` 处理新旧格式的转换，确保向后兼容。

2. **日期范围验证**：日历组件会根据 `startDate` 和 `endDate` 禁用超出范围的日期。

3. **只读模式**：通过 `readonly` 属性可以禁用编辑功能。

4. **餐数计算**：`calculateMealCounts` 函数根据选中的日期和餐次配置计算总餐数。

5. **样式定制**：所有组件使用 scoped CSS，不会影响外部样式。
